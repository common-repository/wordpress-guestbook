== Description ==

WordPress Guestbook is a simple plugin that embeds a guestbook page on your website and ensures it is correctly setup to allow visitor comments.

The plugin itself is very simple and does most of the work utilising WordPress's inbuilt commands to create a guestbook page and validate comment writability.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Upgrade Notice ==

1. Remove /wp-contents/plugins/wordpress-guestbook
2. Unpack the new downloaded version into the same place
